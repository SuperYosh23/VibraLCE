# Changelog

All notable changes to the Viso Design System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of Viso Design System
- Dark theme with Minecraft-inspired green accents
- Glassmorphism effects with backdrop blur
- Comprehensive design token system
- Button components (primary, secondary, ghost, danger)
- Card components (glass and solid variants)
- Form elements (input, select, checkbox, slider)
- Progress indicators (bar and spinner)
- Badge and tooltip components
- Custom scrollbar styling
- Utility classes for common patterns
- Full documentation with examples
- Example HTML file showcasing all components

### Design Tokens
- Color palette with semantic naming
- Typography scale with system font stack
- Spacing system (4px grid)
- Border radius scale
- Shadow elevation system
- Transition timing functions
- Z-index scale

### Components
- `viso-btn` - Button base class with variants
- `viso-card` - Glassmorphism card
- `viso-card-solid` - Solid background card
- `viso-input` - Text input styling
- `viso-select` - Select dropdown
- `viso-checkbox` - Custom checkbox
- `viso-slider` - Range slider with smooth interactions
- `viso-progress` - Progress bar
- `viso-spinner` - Loading spinner
- `viso-badge` - Status badges
- `viso-tooltip` - Hover tooltips
